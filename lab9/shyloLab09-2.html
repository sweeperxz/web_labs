Розділ 1: Вступ до HTML

HTML (HyperText Markup Language) є основною мовою розмітки для створення веб-сторінок. Вона дозволяє визначати структуру документа і взаємодію з іншими елементами.

Розділ 2: Основні теги

HTML містить багато тегів для різних частин веб-сторінок, таких як заголовки, абзаци, списки, таблиці тощо. Важливо правильно використовувати ці теги для створення зрозумілої структури.
